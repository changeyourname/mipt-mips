MIPT-MIPS

Functional and performance simulators for MIPS-based microprocessor

Download this project as a .zip file Download this project as a tar.gz file

This project is a part of Intel lab's activity at Moscow Institute of Physics and Technology (MIPT). The project is being developed by a group of students under the guidance of Intel employees.

The aim of the project is to teach the students the computer architecture through development of a microprocessor implementing the MIPS instruction set in both functional and performance simulators.

More information about MIPT Intel lab history and goals is available on website (in Russian).

How to play audio narrations in Powerpoint

Some presentations contain narrations, they are marked with phrase "Rus/Eng audio". You can play them in MS Powerpoint as an ordinary video. To start playing select the following values in the checkboxes on Slide Show tab (point 1). Then click From Beginning button (point 2), which starts the presentation with audio and automatic animations from the first foil. You also can start this mode from any slide, pause it and so on. If you want to look the presentation without animations and audio just deselect the checkboxes.

Useful books and materials

Microprocessor Arch/uArch

  • Computer Organization and Design: The Hardware/Software Interface, Patterson D.A., Hennessy J.L., 3rd Edition (Eng) -- the main book for this course. It contains all that you need to know about the computer architecture and MIPS ISA.
  • Computer Architecture Online Course by David Wentzlaff at coursera.org (Eng) -- one of the best online course on Computer Architecture.

Programming

  • C++ manual, Herbert Schildt, 3rd edition (Rus) -- this is the main book for studing C++ in this project. The complexity growths from the zero level, thus, even if you don't know C you can learn C++ using this book.
  • The C programming language, Kernighan B.W., Ritchie D.M. (Rus) -- although we are working mostly with C++, some elegant solutions can be made with "pure C", which means "K&R C".
  • Asim: A Performance Model Framework (Eng) is a paper about framework for CPU performance modeling, including features that we're reusing for MIPT-MIPS simulator.

MIPS

  • MIPS Instruction Reference -- the detailed list of MIPS instructions.
  • MIPS Instruction Converter -- the good tool investigate MIPS instruction encoding.
  • MARS Syscalls Funstions -- system calls that are implemented in MARS MIPS simulator and we re-use

About us

Brief changes history

  • 2009/2010: Alexandr Titov and Nikolay Kosarev have started "MDSP simulator" project to create light version of functional and clock-precise simulators of a microprocessor optimized for digital signal processing (DSP) for MIPT students education.
  • 2010/2011: MDSP simulator has been continued by Alexandr Titov, Yuriy Baida and Grigory Rechistov.
  • 2011/2012: New mentor team consisted of Alexander Titov and Pavel Kryukov has taken direction to making course more about theory than simulation.
  • 2012/2013: Complicated MDSP ISA has been changed to more common MIPS and project got its current pun name "MIPT-MIPS".
  • 2013/2014 — 2014/2015: Alexandr and Pavel continued MIPT-MIPS project, adding new lectures material and assignments.
  • 2015/2016: Project moves from GoogleCode SVN to GitHub. The new mentors team consists of Pavel Kryukov and Igor Smirnov.
